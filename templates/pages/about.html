About Pothole Monitor

Pothole Monitor is a website where you can report any pothole you find in Purworejo, Central Java.

FAQ
  • Do I need to signup?
    You don't need to signup if you only want to view Pothole. However, if you want to report Potholes, then signingup is the only way to go.
  • What is 'Response Time Needed'?
    It indicates the emergency of the Pothole. For example, if it is huge that it's blocking the entire road, you should select '< A Day' in Response Time Needed.
